1目的

本規約は、Ballet Studio CONFY（以下「当スタジオ」といいます。）の会員（以下「会員」といいます。）による施設およびレッスン利用に関する条件を定めるものです。

2会員資格・入会手続

  1. 入会希望者は、当スタジオ所定の申込フォームに必要事項を記入し、本規約およびプライバシーポリシーに同意のうえ申し込むものとします。
  2. 入会は、当スタジオの承認をもって成立します。虚偽申告、不適切行為その他当スタジオが会員として不適当と判断する事情が判明した場合、当スタジオは入会をお断りし、または承認を取り消すことがあります。
  3. 未成年者の入会には、保護者の同意を要します。

3料金・支払い

  1. 入会金は 15,000円（税込）とします。月謝その他の料金は、当スタジオが別途定める金額によるものとし、クラス種別、受講回数その他の条件に応じて異なります。
  2. 月謝および設備利用料は、毎月25日までに翌月分を前納するものとし、当スタジオ指定の決済システムにより支払うものとします。なお、決済システム導入前は銀行振込により支払うものとし、振込手数料は会員の負担とします。
  3. 期日までに入金が確認できない場合、当スタジオは、レッスン受講の停止、振替予約の停止その他必要な措置を講じることができます。なお、2か月以上月謝等の支払いが確認できない場合、当スタジオは会員資格を喪失させ、退会扱いとすることがあります。この場合であっても、既に発生している未払金その他の債務の支払義務は消滅しません。
  4. 一旦納入された料金は、当スタジオの都合による休講その他当スタジオが返金を要すると判断した場合を除き、返金しません。
  5. 料金の見直しを行う場合、当スタジオは、原則として1か月以上前に通知します。

4欠席・振替・遅刻

  1. 欠席連絡は、レッスン開始時刻の1時間前までに、メール（balletstudio@confy.co.jp）または公式LINEにて行うものとします。なお、大人会員の欠席連絡は任意としますが、振替を希望する場合は、本項の期限までに連絡を行うものとします。
  2. 振替は、月2回まで、かつ振替期限は翌月末までとします。振替は、空席があり、かつ年齢、レベル、安全面その他の事情に照らして講師が適当と認めるクラスに限り可能とします。振替希望は所定の方法により事前に申請するものとし、満席その他運営上の理由により希望日に振替できない場合があります。
  3. レッスン開始1時間前以降の欠席連絡、または無断欠席は、振替対象外とします。
  4. 遅刻した場合の途中参加可否は、インストラクターが安全面を踏まえて判断するものとします。必要に応じて、見学のみの案内とする場合があります。

5休会・退会

  1. 休会は1か月単位とし、最長3か月までとします。
  2. 休会申請は、前月10日までに、当スタジオ所定の方法（休会申請フォーム）により行うものとします。
  3. 休会中は、レッスン受講および予約はできません。
  4. 休会費は、設備維持費およびシステム管理費として、月額2,000円（税込）とします。
  5. 退会は、当月10日までの申請により当月末退会とし、10日を過ぎた申請は翌月末退会扱いとします。
  6. 退会手続は、当スタジオ所定の「退会届フォーム」から行うものとし、口頭、電話、SNSのDMその他これに類する方法による申請は無効とします。退会申請は、当該フォームの送信完了時点で当スタジオに到達したものとします。ただし、記載不備その他補正が必要な場合には、当該不備が解消された日を申請日とすることがあります。
  7. 月途中の退会による返金は行いません。退会時に未納の月謝、設備利用料その他会員が当スタジオに対して負担する債務がある場合、会員はこれを直ちに精算するものとします。退会または会員資格喪失後も、当該支払義務は消滅しません。
  8. 退会後の再入会には、入会金が必要となる場合があります。なお、個人情報は法令に基づく範囲で一定期間保管します。

6クラス編成および進級

  1. ジュニアクラスの編成は、原則として年齢および学年を基準として行います。
  2. ただし、技術レベル、身体的成長、理解度、レッスン態度、安全面その他の事情を総合的に考慮し、講師の判断により、進級、クラス変更または在籍クラスの調整を行う場合があります。
  3. クラス編成および進級は、原則として当スタジオが定める時期に実施しますが、必要に応じて年度途中で変更する場合があります。
  4. クラス編成、進級およびクラス変更に関する最終判断は、当スタジオおよび講師の裁量によるものとします。

7服装・持ち物・マナー

  1. 会員は、クラス指定の服装を着用し、アクセサリー類を外し、髪をまとめた状態で受講するものとします。
  2. ジュニアクラスに関しては、当スタジオ指定のレオタードを着用するものとします。指定用品は、入会時に当スタジオにて購入するものとします。
  3. スタジオ内での飲食は、水分補給を除き禁止とします。ゴミは各自で持ち帰るものとします。
  4. 他の会員、講師またはスタッフに対する迷惑行為、ハラスメント、無断撮影その他当スタジオの秩序を乱す行為を禁止します。
  5. 肌にオイル、クリーム等を塗布している場合、床面が滑りやすくなり危険であるため、ストレッチ等の際は、素肌が床に直接触れないよう、タオルやヨガマット等を敷くものとします。
  6. 整髪料を使用している場合、髪が床に触れると滑りの原因となるため、タオル等を敷くなど必要な対策を講じるものとします。なお、整髪は洗面所内のみ可能とします。
  7. 自転車で来場する場合、スタジオ周辺への駐輪を禁止します。近隣の駐輪場を利用するものとします。

8安全・免責

  1. 会員は、自己の体調および能力に応じてレッスンを受講するものとし、体調不良、既往症その他安全面に関わる事情がある場合は、事前に当スタジオへ申告するものとします。
  2. レッスン中の事故、怪我、盗難、紛失、私物破損その他の損害については、当スタジオの故意または過失により生じた場合を除き、当スタジオは責任を負いません。会員の自己管理に属する体調悪化、既往症の発症、会員相互間のトラブルその他当スタジオの責めに帰すべき事由によらない損害についても同様とします。
  3. 未成年会員の送迎および帰宅時の安全確保は、保護者の責任とします。
  4. 当スタジオは、必要に応じて、スポーツ安全保険等の任意保険への加入を案内する場合があります。

9撮影・掲載・SNS運用および勧誘の禁止

  1. 当スタジオは、広報または記録の目的で、レッスン、公演その他の活動を写真または動画で撮影し、SNS、YouTube、公式ホームページ、印刷物その他の媒体に掲載することがあります。肖像の利用可否は、申込フォームにおいて選択できるものとします。
  2. 掲載に同意した場合、会員の顔、氏名またはハンドルネーム等が、地域またはオンライン上で認知される可能性があります。会員は、第三者からのDM、コメント、接触等に対し、個人情報を不用意に開示しないよう十分注意するものとします。
  3. 会員は、スタジオ内外（SNS、DM、イベント会場、待合スペース等を含みます。）において、次の各号に定める勧誘、営業その他これに類する行為を行ってはなりません。
    1. 会員、保護者、講師またはスタッフに対する宗教、政治、ネットワークビジネス、販売等の勧誘
    2. イベント、他教室、外部サービス等への無断誘導
    3. 当スタジオ名、講師名、出演動画等の信用を利用した営業、スカウトまたは仲介
    4. 会員情報（氏名、連絡先、在籍状況、出演スケジュール等）の無断収集または第三者提供
  4. 当スタジオが紹介、調整または関与した出演依頼、仕事依頼、メディア取材その他これらに類する案件については、事務局（balletstudio@confy.co.jp）を窓口とします。会員は、当スタジオが紹介または調整した案件について、当スタジオを介さず第三者と直接交渉し、契約し、または受任してはならないものとします。ただし、会員が当スタジオと無関係に独自に受ける案件まで一律に制限するものではありませんが、その場合であっても、当スタジオ名、講師名、所属、出演歴、出演動画等の信用を無断で利用してはならないものとします。
  5. 迷惑行為、つきまとい、悪質な勧誘その他これらに類する行為が認められる場合、当スタジオは、警告、出入禁止、退会その他必要な措置を講じ、必要に応じて警察その他関係機関へ通報することがあります。
  6. 会員による撮影またはSNS投稿は、他の会員のプライバシーおよび安全に十分配慮し、スタッフの許可または指示に従うものとします。特に、レッスン中に撮影された写真または動画のSNS投稿は、当スタジオの事前の許可がある場合を除き、禁止とします。

10臨時休講・代替対応

  1. インストラクターの体調不良、施設の不具合、交通機関の運休、荒天、災害、行政要請、感染症その他やむを得ない事情により、臨時休講またはスケジュール変更を行う場合があります。
  2. 前項の場合の対応の優先順位は、次のとおりとします。
    1. 振替レッスンの提供
    2. オンラインレッスンへの振替
    3. 同月内の追加クラスの提供
    4. 日割相当額の返金または翌月充当（当スタジオ都合により、前各号の対応ができない場合に限ります。）
  3. 実施可否は、原則としてレッスン開始2時間前までに、公式SNS、メールその他当スタジオ所定の方法により告知します。ただし、緊急時はこの限りではありません。

11所属・掛け持ちに関するルール

  1. 成人会員（18歳以上の会員）は、他のバレエ教室またはスタジオの受講（掛け持ち）をすることができます。ただし、安全上、運営上、出演上または指導上の支障がある場合には、当スタジオは、レッスン受講、出演、配役その他必要な範囲で制限または調整を行うことがあります。
  2. 成人会員が掛け持ちをする場合の条件は、次のとおりとします。
    1. 他教室の受講、出演予定その他当スタジオの指導または出演活動に影響する事項については、開始前または判明次第、速やかに事務局へ申告すること
    2. 当スタジオの発表会、コンクール関連の合同稽古および本番を優先すること
    3. 第9条に定める勧誘、引き抜き、営業その他これらに類する行為を行わないこと
    4. 同一期間に重複する本番や、同一または類似演目への出演については、講師の判断に従うこと
  3. ジュニアクラス在籍者および高校生以下の会員の掛け持ちは、原則として禁止します。ただし、学校授業、学内公演、一般のオープンクラス、短期ワークショップその他当スタジオが相当と認めた場合は、事前申請のうえ例外として認めることがあります。
  4. コンクール、発表会、外部公演その他これらに類する場面における所属表記は、「Ballet Studio CONFY」を基本とし、他教室名の併記または名義使用は、当スタジオの事前承認がある場合を除き認めません。

12発表会・コンクール等への参加

  1. 発表会、コンクールその他これらに類する催事への参加条件は、当スタジオ在籍会員であることとし、出席状況、技術、健康状態その他の事情を踏まえ、講師の審査または推薦により参加可否、配役および演目を決定します。
  2. 参加希望者は、当スタジオ所定の申請フォームにより、申請締切までに必要事項を提出するものとします。申請内容の変更または追加は、原則として締切後は受け付けません。
  3. 選考にあたり、動画提出、リハーサル審査その他の方法を実施する場合があります。芸術的判断その他運営上の必要により、配役、出演順または構成を変更する場合があります。
  4. 参加費、衣装費、指導料、追加リハーサル費、音源編集費、引率費、交通費、宿泊費、保険料その他の実費は、参加者の負担とし、支払期限までに前納するものとします。
  5. 申請確定後の自己都合によるキャンセルについては、原則として返金しません。当スタジオ都合で中止となり、振替または延期等の代替措置を提供できない場合は、未提供分相当額を按分返金し、または翌月以降に充当します。
  6. 本番前リハーサル、合同稽古その他大会等に関連する重要日程については、最優先で参加するものとします。度重なる欠席または遅刻がある場合、出演辞退、配役変更その他必要な調整の対象となることがあります。
  7. 舞台、リハーサルまたは本番の写真および映像の撮影、配信、販売等に関する権利は、当スタジオまたは主催者が管理するものとし、会員はこれに同意するものとします。なお、個別の同意取得が必要な場合は、この限りではありません。
  8. 出場先の大会主催者または会場の規約（安全、著作権、肖像権、年齢区分等を含みます。）を遵守するものとし、違反による不利益は本人の負担とします。
  9. 主催者、メディア、スカウト等との連絡のうち、当スタジオが紹介、調整または関与した案件については、原則として事務局（balletstudio@confy.co.jp）を窓口とします。参加者が当スタジオと無関係に受ける連絡まで一律に制限するものではありませんが、当スタジオ名、講師名、所属、出演歴その他当スタジオの信用を用いて直接交渉し、契約し、または受任する場合には、事前に事務局の承認を要するものとします。
  10. 移動、宿泊または待機中の盗難、事故、体調不良等については、本人または保護者の管理責任とし、当スタジオに故意または重過失がある場合を除き、責任を負いません。必要に応じて、任意保険への加入を推奨します。
  11. 大会等における所属名称は「Ballet Studio CONFY」とし、他教室名の併記または名義使用は、第11条に定めるルールおよび当スタジオの事前承認に従うものとします。
  12. 出演情報、写真または動画は、当スタジオのSNS、YouTube、Webサイトその他の媒体において告知または掲載することがあります。なお、これらの取扱いは、プライバシーポリシーおよび申込時の同意内容に従うものとします。

13施設・設備利用

  1. 会員の故意または過失により設備を破損または汚損した場合、当スタジオは、その修理費その他実費相当額の負担を求めることができます。
  2. 災害、感染症、天候その他の不可抗力により、休講、オンライン振替またはスケジュール変更を行う場合があります。

14個人情報の取扱い

会員の個人情報の取扱いについては、当スタジオが別途定める プライバシーポリシー に従うものとします。

15反社会的勢力の排除および迷惑行為の禁止

  1. 会員が反社会的勢力に該当し、またはこれと関係を有することが判明した場合、当スタジオは、入会を拒否し、または退会とすることができます。
  2. 他の会員、講師、スタッフ、近隣住民等に対する迷惑行為、秩序を乱す行為その他当スタジオが不適切と判断する行為があった場合、当スタジオは、警告、利用制限、入会拒否または退会その他必要な措置を講じることができます。

16規約の改定

当スタジオは、必要に応じて本規約を改定することがあります。重要な変更を行う場合は、Webサイト、SNS、メールその他当スタジオが適当と認める方法により告知します。

※ 本規約は、法令や運営方針により変更となる場合があります。
